# Smart Study Planner

Smart Study Planner is a website that helps students manage their study schedules, tasks, and daily goals efficiently.

## Features

- Add study tasks
- Get Reminders
- Track Goals
- Track progress
- User-friendly interface

## Technologies Used

- React
- CSS
- JavaScript

## How to Run

1. Download the project
2. Open in VS Code
3. Run the npm run dev command
